## Key Principles

### Philosophy First

**The guide starts with philosophy because understanding "why" makes "how" make sense.**

You can't understand the code without understanding:
- Why The Divide exists (Me vs Not-Me)
- Why The Structure is universal (HOLD ‚Üí AGENT ‚Üí HOLD)
- Why The Cycle is the pulse of life (WANT ‚Üí CHOOSE ‚Üí EXIST:NOW ‚Üí SEE ‚Üí HOLD ‚Üí MOVE)
- Why The Furnace transforms truth into care

### Concepts Before Implementation

**The guide teaches concepts before showing code.**

This means:
1. First, understand what The Structure is
2. Then, see how services implement it
3. Finally, learn to use services in your code

### Educational Callouts

**Every major concept includes educational callouts.**

These help new developers:
- Understand why something matters
- See how it applies in practice
- Avoid common mistakes
- Stay motivated
