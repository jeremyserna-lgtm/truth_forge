## What This Pattern Is

The Resonant Learning Pattern is a **meta-pattern**: a pattern that improves itself through understanding.

### The Core Loop

```
Detect → Sense Resonance → Understand → Learn → Adapt → Detect (better)
```

This creates a **self-improving feedback loop** where:
- The system detects significant moments
- The system senses which moments resonate (align with essence)
- The system understands what resonates and why
- The system learns which patterns work best
- The system adapts to focus on what resonates
- The system detects better moments (more resonant)
- The cycle continues, improving each iteration
