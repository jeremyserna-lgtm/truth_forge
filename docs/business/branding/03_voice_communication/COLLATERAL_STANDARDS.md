## Business Cards

### Format
- Size: 3.5" × 2" (standard US)
- Paper: Uncoated, heavy stock (14pt minimum)
- Finish: Soft touch matte OR letterpress texture
- Color: Dark mode preferred (charcoal background)

### Layout

**Front:**
```
┌─────────────────────────────────────────┐
│                                         │
│  [LOGO MARK]                            │
│                                         │
│  JEREMY SERNA                           │
│  Founder & Architect                    │
│                                         │
│  jeremy@truthengine.co                  │
│  +1 (XXX) XXX-XXXX                      │
│                                         │
│  TRUTH ENGINE                           │
│  UNIT 001                               │
│                                         │
└─────────────────────────────────────────┘
```

**Back:**
```
┌─────────────────────────────────────────┐
│                                         │
│                                         │
│        The Source of All Systems        │
│                                         │
│        truthengine.com                  │
│                                         │
│                                         │
└─────────────────────────────────────────┘
```

### Typography
- Name: Instrument Serif, 12pt
- Title: JetBrains Mono, 9pt
- Contact: JetBrains Mono, 8pt
- Brand/Unit: JetBrains Mono, 8pt
- Tagline: Instrument Serif, 10pt, italic

- Size: 8.5" × 11" (US Letter)
- Margins: 1" all sides

```
─────────────────────────────────────────────────
```

### Footer Layout
```
─────────────────────────────────────────────────
TRUTH ENGINE LLC · Denver, Colorado · truthengine.com
```

### Typography
- Body: Inter, 11pt, 1.5 line spacing
- Header/Footer: JetBrains Mono, 9pt

## Presentation Templates

### Slide Deck Standards

**Master Slide:**
```
┌─────────────────────────────────────────────────────────┐
│ [LOGO]                              [Page Number]       │
│                                                         │
│                                                         │
│                   [CONTENT AREA]                        │
│                                                         │
│                                                         │
│                                                         │
│─────────────────────────────────────────────────────────│
│ TRUTH ENGINE · The Source of All Systems                │
└─────────────────────────────────────────────────────────┘
```

**Title Slide:**
```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│                        [LOGO]                           │
│                                                         │
│              PRESENTATION TITLE                         │
│              Subtitle or Date                           │
│                                                         │
│                   [Texture Background]                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### Typography
- Titles: Instrument Serif, 44pt
- Subtitles: JetBrains Mono, 24pt
- Body: Inter, 20pt
- Captions: JetBrains Mono, 14pt

### Color Usage
- Background: Charcoal (#1A1A1A) or Warm White (#F5F0E6)
- Text: Inverse of background
- Accent: Brand-specific color for highlights

### Slide Types

1. **Title** — Logo, presentation name, date
2. **Section** — Section title centered, textured background
3. **Content** — Standard body content
4. **Data** — Charts, graphs, statistics
5. **Quote** — Testimonial or key quote
6. **Image** — Full-bleed or framed image
7. **Comparison** — Two-column layout
8. **End** — Thank you, contact information

## Proposal Documents

### Structure

1. **Cover Page**
   - Logo (large, centered)
   - Proposal title
   - Client name
   - Date
   - Unit designation (if applicable)

2. **Table of Contents**
   - Numbered sections
   - Page numbers

3. **Executive Summary**
   - 1 page maximum
   - Problem → Solution → Investment

4. **Body Sections**
   - Clear headers (Instrument Serif)
   - Body text (Inter)
   - Data/specs (JetBrains Mono)

5. **Investment/Pricing**
   - Clear pricing tables
   - Terms
   - Timeline

6. **Next Steps**
   - Call to action
   - Contact information

### Formatting
- Headers: Instrument Serif
- Body: Inter, 11pt
- Technical: JetBrains Mono
- Margins: 1" all sides
- Page numbers: Bottom right, JetBrains Mono

|-------|------|-----|
| Black | 0/0/0/100 | Primary text, marks |
| Charcoal | 0/0/0/90 | Backgrounds |

## Quality Checklist

Before producing any collateral:

- [ ] Correct logo version used
- [ ] Typography matches brand standards
- [ ] Colors match brand palette
- [ ] Texture level appropriate for brand
- [ ] Contact information current
- [ ] Legal entity correct (LLC name)
- [ ] Unit designation formatted correctly
- [ ] Tagline matches current version
- [ ] Proofread for errors
- [ ] File named according to convention
