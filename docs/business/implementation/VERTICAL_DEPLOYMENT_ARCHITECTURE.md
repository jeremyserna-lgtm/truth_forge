1.  DISCOVERY: Understand firm's workflows, use cases, integration needs

3.  PROMPT CUSTOMIZATION: Craft system prompt for firm's specific context

4.  HARDWARE CONFIGURATION: Prepare Mac Studio with model and prompt

5.  DEPLOYMENT: Ship or install hardware at client site

6.  INTEGRATION: Connect to firm's systems (document management, email, etc.)
