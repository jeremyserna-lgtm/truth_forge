# Truth Forge Website Structure

**Purpose**: Complete information architecture, page structure, and navigation design for truth-forge.ai

**Status**: Proposed Structure  
**Last Updated**: 2026-01-27

---

## Executive Summary

This document outlines the complete website structure, maps current pages, identifies missing elements, and proposes an improved information architecture that is attractive, easy to understand, navigable, and honors our quality standards.

**Core Principle**: The website should feel like **a physical space**—not a digital interface. Visitors should feel they've entered somewhere crafted, warm, and permanent.

## Page Layout Standards

### Standard Page Template

Every page follows this structure:

```
┌─────────────────────────────────────────────────────────────┐
│ NAVIGATION (sticky)                                         │
├─────────────────────────────────────────────────────────────┤
│                    PAGE HERO                                │
│          [Headline - Instrument Serif]                      │
│          [Introduction - Inter]                              │
├─────────────────────────────────────────────────────────────┤
│                    MAIN CONTENT                              │
│          [Sections with H2 headings]                        │
│          [Cards, grids, text blocks]                        │
├─────────────────────────────────────────────────────────────┤
│                    CTA SECTION (if applicable)              │
│          [Call to action]                                    │
├─────────────────────────────────────────────────────────────┤
│                    FOOTER                                    │
│          [Brand family, links, legal]                      │
└─────────────────────────────────────────────────────────────┘
```

### Section Patterns

**Hero Section**:
- Full-width, centered content
- Large headline (Instrument Serif)
- Tagline/description (Inter)
- Optional CTA button

**Content Sections**:
- Max-width container (1200px)
- H2 section headings (Instrument Serif, centered)
- Section intro text (Inter, centered, max-width 700px)
- Content cards/grids
- Alternating left/right layouts for variety

**CTA Sections**:
- Full-width, centered
- Clear value proposition
- Prominent button

## Quality Standards Compliance

### Performance
- ✅ Lighthouse score: 90+ on all metrics
- ✅ First Contentful Paint: < 1.5s
- ✅ Time to Interactive: < 3s
- ✅ Image optimization: WebP with fallback

### Accessibility
- ✅ WCAG 2.1 AA compliance
- ✅ Keyboard navigation
- ✅ Screen reader support
- ✅ Color contrast: 4.5:1 minimum

### SEO
- ✅ Page titles: `[Page Name] | Truth Forge`
- ✅ Meta descriptions: 150-160 characters
- ✅ Heading hierarchy: Single H1, logical H2-H6
- ✅ Alt text on all images

### Design System
- ✅ Colors: Design tokens (CSS variables)
- ✅ Typography: Instrument Serif, JetBrains Mono, Inter
- ✅ Spacing: Design tokens (--space-*)
- ✅ Components: Reusable patterns
