## Component Standards

**See**: [COMPONENTS.md](COMPONENTS.md)

**Key Requirements**:
- All components follow HOLD:AGENT:HOLD pattern
- Use design system tokens (colors, typography, spacing)
- Responsive by default
- Accessible (WCAG 2.1 AA minimum)

---

## Routing Standards

**See**: [ROUTING.md](ROUTING.md)

**Key Requirements**:
- Clean, semantic URLs
- Consistent page structure
- Navigation hierarchy
- SEO-friendly patterns

---

## Performance Standards

**See**: [PERFORMANCE.md](PERFORMANCE.md)

**Key Requirements**:
- Lighthouse score: 90+ on all metrics
- First Contentful Paint: < 1.5s
- Time to Interactive: < 3s
- Image optimization: WebP with fallback

## Content Standards

**See**: [CONTENT.md](CONTENT.md)

**Key Requirements**:
- Truth Forge voice (warm, authoritative, human)
- Framework-aligned language
- Stage 5 audience appropriate
- Clear, direct communication
